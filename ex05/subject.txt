Exercise 05: Harl 2.0
Directory: ex05/
Files to Submit: Makefile, main.cpp, Harl.{h, hpp}, Harl.cpp
Forbidden: None

• "DEBUG" level:
Example: "I love having extra bacon for my 7XL-double-cheese-triple-pickle-special-
ketchup burger. I really do!"
• "INFO" level:
Example: "I cannot believe adding extra bacon costs more money. You didn’t put
enough bacon in my burger! If you did, I wouldn’t be asking for more!"
• "WARNING" level: 
Example: "I think I deserve to have some extra bacon for free. I’ve been coming for
years, whereas you started working here just last month."
• "ERROR" level:
Example: "This is unacceptable! I want to speak to the manager now."
You have to create a Harl class with the following private member functions:
• void debug( void );
• void info( void );
• void warning( void );
• void error( void );
public member function
depending on the level passed as a parameter:
void	complain( std::string level );

The goal of this exercise is to use pointers to member functions.

Create and turn in tests to show that Harl complains a lot.
